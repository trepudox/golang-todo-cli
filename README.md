# golang-todo-cli

pra facilitar o uso, crie um alias com o caminho de 'main.go' pra rodar o comando:

```bash
alias todo="go run ~/DEV/Golang/golang-todo-cli/main.go"
```

ele é temporário, sendo necessário rodar a cada nova sessão no terminal

pra criar esse alias de forma permanente adicione-o no ~/.bashrc
